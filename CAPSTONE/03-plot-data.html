Installing the required packages.

In this lab, we will be using R to plot data onto maps. Before gettign started, we need to install a few packages. This can be done with command line:

install.packages("PACKAGE_NAME")

Or using the graphical interface in RStudio: Tools -> Install packages.

The packages that we will need are: * ggplot2 * maps * mapdata * RColorBrewer

You can install all packages in one line of code with:

install.packages( c("ggplot2", "maps", "mapdata", "RColorBrewer") )

Before we can start using the functions provided by these packages (but after we install them), we need to load the packages in memory (this step needs to be done again for every new R session):

library(ggplot2)
library(maps)
library(mapdata)
library(RColorBrewer)

Examples of maps:

Lets start with a map of the entire world:

world_map <- map_data("world")
ggplot(world_map, aes(x = long, y = lat, group = group)) +

And a map of the US, without any state/county borders drawn:

usa_map <- map_data("usa")
ggplot(usa_map, aes(x = long, y = lat, group = group)) +
  geom_polygon(fill="lightgray", colour = "white")

Adding the states borders and changing the aspect ratio a notch:

usa <- map_data("state")

usa_map <- ggplot(data=usa, mapping=aes(x=long, y=lat, group=group)) + 
  coord_fixed(1.3) + 
  geom_polygon(color="black", fill="grey")

plot(usa_map)

Zooming on the state of Mississippi + county borders:

usa <- map_data('state')
ms <- subset(usa, region=="mississippi")

counties <- map_data("county")

  geom_polygon(color="black", fill=NA)


plot(myMap)

Adding points based on geographical coordinates.

In this first exercise, we will add to a usmap a number of points defined by geographical coordinates (longitude and latitude). To illustrate this principle, I will plot the location of the 3 places that Ive lived in (in the US) with dots of size proportional to the amount of time that Ive spent in each city.

The three cities are:

  • Bloomington, IN (86.52 West, 39.15 North) - 7 years
  • Scottsdale, AZ (111.89 West, 33.46 North) - 1 year
  • Starkville, MS (88.73 West, 33.35 North) - 6 years

Lets start by creating a data.frame containing all this information:

places <- data.frame(
  longitude = c(-86.52, -111.89, -88.73),
  latitude = c(39.15,  33.46, 33.35),
  years = c(7, 1, 6),
  labels = c("Bloomington, IN", "Phoenix, AZ", "Starkville, MS")
)

Note how longitudes West of the Greenwitch meridian are negative values.

usa <- map_data("state")

usa_map <- ggplot(data=usa, mapping=aes(x=long, y=lat, group=group)) + 
  coord_fixed(1.3) + 
  geom_polygon(color="black", fill="gray") + 
  geom_polygon(data=usa, fill=NA, color="white") + 
  geom_polygon(color="black", fill=NA)

mapWithCities <- usa_map + geom_point(data=places, aes(x=longitude, y=latitude, size=years), col="red", inherit.aes = F)

mapWithCities + geom_text(data = places, aes(label = labels, x = longitude+2, y = latitude+1), inherit.aes = F, col="steelblue")

Take a minute to generate a map of the places youve lived in (or places you like most in the US). Try to adjust the location of labels so that they are nicely positioned relatively to the dots on the map.

Saguaro data
