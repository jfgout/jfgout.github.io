A practical example of loops

Using loops to compute the correlation between OD and concentration.

I have created 5 result files (csv files with two columns: concentration and OD) in the folder: C:/lab/MSSTATE/Class/CAPSTONE/2024/repo/files/OD_concentration/ on my laptop. The goal is to analyse all these files, using a loop. For each file, I will load the data in R, compute the correlation between OD and concentration, plot the corresponding graph and display the value for R-square.

You can download these 5 files here:

  • chelsea.csv
  • erin.csv
  • hamna.csv
  • kristin.csv
  • lori.csv

And here is the code to analyze the data:

# First, I set my working directory to the location where the files are.
setd("C:/lab/MSSTATE/Class/CAPSTONE/2024/repo/files/OD_concentration/")

vFiles <- list.files(path = ".", pattern = "*.csv")

i <- 1
nbFiles <- length(vFiles)
while( i <= nbFiles ){
  fName <- vFiles[i]
  t <- read.csv(fName)
  summaryCor <- summary(lm(t$OD~t$concentration))
  rsquare <- round( as.numeric(summaryCor$adj.r.squared) , 2 )
  mainTitle <- paste(fName, " - R-square: ", rsquare, sep = "")
  plot(t$OD~t$concentration, xlab = "Food dye concentration", ylab = "Optical Density", main = mainTitle)
  abline(lm(t$OD~t$concentration), col = "red")
  #print(gp)
  readline(prompt="Press [enter] to continue")
  i <- i + 1  
}

To make pretty graphs, we can use the package ggplot. You can install packages with the command line:

install.packages(package_name)

Or use the graphical interface in R studio (Menu Tools –> Install package)

library(ggplot2)

# First, I set my working directory to the location where the files are.
setd("C:/lab/MSSTATE/Class/CAPSTONE/2024/repo/files/OD_concentration/")

vFiles <- list.files(path = ".", pattern = "*.csv")

i <- 1
nbFiles <- length(vFiles)
while( i <= nbFiles ){
  fName <- vFiles[i]
  t <- read.csv(fName)
  summaryCor <- summary(lm(t$OD~t$concentration))
  rsquare <- round( as.numeric(summaryCor$adj.r.squared) , 2 )
  mainTitle <- paste(fName, " - R-square: ", rsquare, sep = "")
  gp <- ggplot(data = t, aes(x = concentration, y = OD)) +
    geom_smooth(method = "lm", se=FALSE, color="red") +
    ggtitle(mainTitle) +
    geom_point()
  print(gp)
  readline(prompt="Press [enter] to continue")
  i <- i + 1  
}
